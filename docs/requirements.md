# 产品需求文档 - 新疆高铁国际旅行社网站

## 项目概述

新疆高铁国际旅行社品牌展示网站，向客户传递旅行社相关信息，建立品牌信任感，引导客户咨询。

## 目标受众

- 国内散客 / 家庭游客
- 高端定制游客
- 企业 / 团体客户

## 核心目标

**品牌展示为主**：展示旅行社形象、路线、口碑，让访客建立信任后线下或电话咨询。

## 一期功能

| 优先级 | 页面 | 核心内容 |
|--------|------|----------|
| P0 | 首页 | 视频背景 Hero、品牌简介、精选路线预览、信任背书、CTA |
| P1 | 关于我们 | 公司介绍、发展历程时间线、核心优势、资质证书 |
| P2 | 路线推荐 | 分类筛选、路线卡片（图+标题+天数+亮点+价格） |
| P3 | 登录 | 手机号验证码登录/注册 |

## 二期预留

- 联系我们（地址、电话、微信、留言表单）
- 客户评价（评价+旅行照片墙）
- 新闻动态（公司新闻、旅游资讯）
- 常见问题（FAQ 折叠面板）

## 非功能性需求

- 移动端响应式适配
- 页面加载速度快（纯静态）
- 设计风格高端、地域文化融合
- 预留视频背景位置
